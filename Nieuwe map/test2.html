Over mezelf

Ten eerste zal ik mezelf voorstellen met behulp van de sleutelvaardigheden die we gezien hebben tijdens de les. Ik geef mezelf een score op 100%. Deze score is een combinatie van hoe ik denk dat ik ben en hoe anderen mij ervaren op deze vaardigheden. Vanonder op de website vind je een verzameling van foto's van zaken, personen, plaatsen die ik in de loop van mijn leven belangrijk vind of heb gevonden. De reden waarom ik voor een website gekozen heb is omdat ik hiervoor informatica gestudeerd heb.
